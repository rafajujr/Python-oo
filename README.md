🐍 Python OO - Programação Orientada a Objetos em Python

Projeto desenvolvido para demonstrar os principais conceitos de Programação Orientada a Objetos (POO) utilizando a linguagem Python.

📖 Sobre o Projeto

Este repositório contém exemplos práticos dos conceitos fundamentais da Programação Orientada a Objetos, permitindo compreender como modelar sistemas de forma organizada, reutilizável e escalável.

Através deste projeto, é possível aprender e praticar:

Classes e Objetos
Atributos e Métodos
Construtores (__init__)
Encapsulamento
Herança
Polimorfismo
Abstração
Composição
Boas práticas de desenvolvimento com Python
🚀 Tecnologias Utilizadas
Python 3.x

💻 Como Executar
1. Clone o repositório
git clone https://github.com/rafajujr/Python-oo.git
2. Acesse a pasta do projeto
cd Python-oo
3. Execute o arquivo principal
python main.py
📚 Conceitos Demonstrados
Classe
class Pessoa:
    def __init__(self, nome):
        self.nome = nome
Objeto
pessoa = Pessoa("Rafael")
Herança
class Funcionario(Pessoa):
    pass
Polimorfismo
class Animal:
    def falar(self):
        pass

class Cachorro(Animal):
    def falar(self):
        return "Au Au"
